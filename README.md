# Headlines or Hallucinations

A fun quiz game that tests your ability to spot fake news from real headlines.

## How to Play

Visit the game at: https://btennett.github.io/headlines-game/

Can you tell which headlines are real and which are made up?

## Features

- 10 questions per game
- Mix of real 2024-2025 headlines and fakes
- Verify answers with Google search links
- Mobile-friendly design
